# Image

> 图片查看器

基于 Avalonia 12 + .NET 10 构建。

![preview](preview/preview.png)

## 开发

```bash
# 编译
./build.sh

# 运行
./start.sh

# 发布 Release 编译
./build.sh release
```

## Build

```bash
# macOS (.dmg)
./publish.sh osx-arm64

# Mac Intel
./publish.sh osx-x64

# Windows (.exe)
./publish.sh win-x64

# Windows 32 位
./publish.sh win-x86
```
